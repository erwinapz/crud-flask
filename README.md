## Configuracion para levantar el proyecto.

* Clonar el repositorio.
   * ```git clone https://github.com/erwinapz/crud-flask.git```
 
* Entrar al directorio clonado.
  * ```cd crud-flask```

* Instalar Flask si no necesario.
  * ```pip install Flask```

  * ```python3 app.py```
 
* Entrar al navegador.
